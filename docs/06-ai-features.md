# AI Features

SignalIQ contains multiple AI agents.

Data Collection Agent

Collects

- Companies
- Prices
- Financial Statements
- News

Analysis Agent

Generates

- Company summaries
- Financial analysis
- Technical analysis
- Risk analysis

Recommendation Agent

Produces

- Strong Buy
- Buy
- Hold
- Sell
- Strong Sell

News Agent

Performs

- Sentiment analysis
- News summarization
- Trend detection

Chat Agent

Allows users to ask natural language questions about companies and investments.
