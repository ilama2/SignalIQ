# Frontend Specification

Framework

- React
- TypeScript
- Vite

UI

- Tailwind CSS
- shadcn/ui
- Lucide Icons

Charts

- Recharts

Routing

React Router

State Management

React Context

Data Fetching

Axios

---

The UI should feel similar to:

- TradingView
- Bloomberg
- Yahoo Finance

Avoid creating an admin dashboard.

The interface should look like a premium financial application.

Design principles:

- Minimal
- Clean
- Professional
- Data-first
- Responsive

Use reusable components everywhere.

No duplicated code.

Pages should remain lightweight.

Business logic belongs in hooks or services.
